## Context
Surprises during RF fieldwork are costly. A rehearsal surfaces gaps while there is still time.

## Decision
Run a **full internal dry-run** (sample evidence, rehearse data-request turnaround, mock SME interviews) before RF fieldwork.

## Alternatives Considered
**Skip the dry-run** — rejected: no rehearsal. **Partial dry-run** — rejected: misses interview readiness.

## Consequences
Confirmed readiness; minor tidy-ups applied; contributed to the 0-violation outcome.
