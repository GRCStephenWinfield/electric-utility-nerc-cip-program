## Context
Inconsistent or slow responses to RF data requests create audit risk.

## Decision
Route all RF data requests through a **single point of contact (Compliance Manager)** with a tracked turnaround process and BCSI handling.

## Alternatives Considered
**Distributed responses** — rejected: inconsistency. **Ad hoc** — rejected: missed timeframes.

## Consequences
Consistent, timely, BCSI-controlled responses; all requests answered on time.
