## Context
The audit package must contain every RSAW, supporting document, Mitigation Plan, and Self-Report the RF audit team expects, consistently.

## Decision
Assemble a **24-component compliance evidence package** (12 RSAWs + supporting registers + Mitigation Plans + Self-Reports + sign-off) under version and BCSI control.

## Alternatives Considered
**Minimal package** — rejected: risks data-request churn. **Ad hoc assembly** — rejected: inconsistency.

## Consequences
Smooth audit; single source of truth; reconciled evidence.
