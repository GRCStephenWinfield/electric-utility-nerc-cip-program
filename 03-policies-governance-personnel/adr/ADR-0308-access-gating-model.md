## Context
CIP-004 requires authorized access based on need, with training and a PRA completed before access is granted.

## Decision
Gate every access grant on **documented need + completed training + current PRA**, verified quarterly, with 24-hour revocation on termination.

## Alternatives Considered
**Grant then verify** — rejected: risks unauthorized access. **Annual review only** — rejected: misses interim changes.

## Consequences
Strong access posture; closes GAP-05; supports CIP-004 R4/R5 evidence.
